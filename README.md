# node-to-app-compiler

## Getting Started

Two ways to get up and running.

**Docker**

```sh
$ cd <project_directory>
$ docker run -it --rm -v "${PWD}":/app node-to-app-compiler:v0.0.1 <option> <package_name>
```

**Terminal**

```sh
$ chmod +x compiler.sh
$ ./complier.sh <package_name>
```
